# Monorepo Boilerplate

A TypeScript monorepo using **pnpm workspaces**, **Vite React** with **Tailwind CSS** for the frontend, a **Node (TSX) server**, and a **shared TypeScript utility package**.  



## Installation


```bash
# Clone the repository
git clone https://github.com/9ggaurav/Monorepo-Express-React.git

# Navigate to the project directory
cd Monorepo-Express-React

# Install dependencies
pnpm install
``` 

Development
```bash
pnpm dev                     # Both frontend and backend
pnpm dev:web                 # Frontend only
pnpm dev:server              # Backend only
```



Build
```bash
pnpm build
```
